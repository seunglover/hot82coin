해킹과 사기로부터 소중한 자산을 지키는 방법을 알아보세요.

⚠️ 중요: 암호화폐 투자는 높은 위험을 동반합니다. 잃어도 괜찮은 돈으로만 투자하세요. 본 내용은 투자 조언이 아닌 정보 제공 목적입니다.
🔐

기본 보안 수칙

모든 투자자가 반드시 지켜야 할 기본적인 보안 수칙들입니다.

🔑 강력한 비밀번호

• 12자리 이상의 복잡한 비밀번호
• 대소문자, 숫자, 특수문자 조합
• 사이트별로 다른 비밀번호 사용
• 비밀번호 관리자 프로그램 활용

📱 2단계 인증(2FA)

• Google Authenticator, Authy 사용
• SMS보다는 OTP 앱 선택
• 백업 코드 안전한 곳에 보관
• 모든 거래소에서 필수 설정

🌐 안전한 네트워크

• 공용 와이파이에서 거래 금지
• VPN 사용 권장
• HTTPS 사이트만 이용
• 공용 컴퓨터에서 로그인 금지

💾 정기 백업

• 지갑 시드 구문 안전 보관
• 여러 장소에 분산 보관
• 하드웨어 백업 권장
• 정기적으로 복구 테스트

👛

지갑 보안 관리

🔒 하드웨어 지갑 사용

• Ledger, Trezor 등 검증된 제품
• 공식 사이트에서만 구매
• 개봉 상태와 씰 확인
• 펌웨어 정기 업데이트

📱 소프트웨어 지갑

• 공식 앱스토어에서만 다운로드
• 정기적인 앱 업데이트
• 루팅/탈옥된 기기 사용 금지
• 바이러스 검사 정기 실행

🏢 거래소 보관 최소화

• 거래 시에만 거래소 이용
• 평소에는 개인 지갑 보관
• 대량 보관 시 분산 보관
• 정기적으로 출금하기

🔑 개인키 관리

• 개인키 절대 온라인 공유 금지
• 시드 구문 오프라인 보관
• 스크린샷이나 사진 저장 금지
• 물리적 백업 여러 개 보관

🕵️

사기 및 피싱 예방

⚠️ 주요 사기 수법들

📧 피싱 이메일

거래소를 사칭한 가짜 이메일로 로그인 정보를 탈취

🎁 가짜 에어드롭

무료 코인을 준다며 개인키나 시드 구문을 요구

👨‍💼 투자 사기

고수익을 보장하며 투자금을 요구하는 폰지 사기

🌐 가짜 사이트

유명 거래소와 비슷한 URL의 피싱 사이트

✅ 사기 예방 수칙

• URL을 정확히 확인하기
• 이메일 링크 클릭 금지
• 고수익 보장 제안 거부
• 개인정보 요구 시 의심
• 서두르라는 압박 무시

🚨 의심스러운 신호들

• 너무 좋은 수익률 제안
• 유명인 사칭 메시지
• 긴급하다며 서두르기
• 선입금 요구
• 개인키/시드 구문 요청

📊

리스크 관리

💰 자금 관리

• 잃어도 괜찮은 돈으로만 투자
• 대출받아서 투자 절대 금지
• 생활비와 투자금 분리
• 포트폴리오 다양화

📈 감정 관리

• FOMO(놓칠까봐 하는 투자) 주의
• 손절매 라인 미리 설정
• 계획적이고 냉정한 판단
• 소셜미디어 과도한 의존 금지

📚 지속적 학습

• 시장 동향과 기술 발전 추적
• 신뢰할 만한 정보원 확보
• 투자 경험 기록 및 분석
• 전문가 의견 참고 (맹신은 금물)

✅ 보안 체크리스트

안전한 투자를 위해 다음 항목들을 확인해보세요:

강력한 비밀번호와 2단계 인증 설정 완료
하드웨어 지갑 구매 및 설정
시드 구문 오프라인 백업 완료
신뢰할 수 있는 거래소만 이용
피싱 사이트 구별법 숙지
투자 계획과 손절매 라인 설정
정기적인 보안 점검 계획 수립
