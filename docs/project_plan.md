# 프로젝트 계획서

## 프로젝트 개요
- 코인 투자 추천 웹 애플리케이션

## 완료된 작업
- 프로젝트 기본 구조 설정

## 현재 작업
- 플레이라이트 스텔스 MCP 테스트 완료

## 완료된 테스트
- 페이지 열기 (google.com, coupang.com) - 성공
- 스크린샷 촬영 - 성공 (다운로드 폴더에 저장됨)
- 콘솔 로그 수집 - 성공
- 텍스트 입력 (fill) - 성공 (구글, 쿠팡 검색창)
- 버튼 클릭 (click) - 성공
- 키보드 입력 (press_key) - 성공

## 발생한 문제
- 구글: CAPTCHA 감지 (자동화 툴 감지)
- 쿠팡: HTTP2_PROTOCOL_ERROR (사이트 서버 문제)
