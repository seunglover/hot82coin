🏠 일반적인 질문

CryptoPulse는 어떤 서비스인가요? +
CryptoPulse는 해외 주요 거래소(Bybit, CoinGecko)의 실시간 데이터를 활용하여 암호화폐 거래량 순위를 제공하는 서비스입니다. 투자에 도움이 되는 정보와 교육 콘텐츠도 함께 제공합니다.
서비스 이용은 무료인가요? +
네, 완전 무료로 이용 가능합니다. 회원가입이나 로그인 없이도 모든 기능을 자유롭게 사용하실 수 있습니다. 광고 수익으로 서비스를 운영하고 있습니다.
모바일에서도 잘 작동하나요? +
네, 모바일 최적화가 되어있어 스마트폰과 태블릿에서도 편리하게 이용하실 수 있습니다. PWA(Progressive Web App) 기능으로 앱처럼 설치해서 사용도 가능합니다.

📊 데이터 관련

데이터는 얼마나 자주 업데이트되나요? +
코인 데이터는 5분마다 자동으로 업데이트됩니다. 거래량과 가격 변동 등 주요 지표들이 실시간으로 반영되어 최신 시장 상황을 파악할 수 있습니다.
어떤 거래소의 데이터를 사용하나요? +
주로 Bybit과 CoinGecko API를 통해 데이터를 수집합니다. 이들은 전 세계적으로 신뢰받는 플랫폼으로, 정확하고 안정적인 데이터를 제공합니다.
원화 가격은 어떻게 계산하나요? +
실시간 USD/KRW 환율을 적용하여 달러 기준 가격을 원화로 환산합니다. 환율 데이터는 ExchangeRate API를 통해 주기적으로 업데이트됩니다.

⚙️ 기술적 문제

페이지가 로딩되지 않아요 +
브라우저 캐시를 삭제하고 페이지를 새로고침해보세요. 그래도 문제가 지속되면 다른 브라우저나 시크릿 모드를 사용해보시고, 계속 문제가 있다면 문의해주세요.
데이터가 표시되지 않아요 +
네트워크 연결 상태를 확인해주세요. 일시적인 API 서버 문제일 수 있으니 잠시 후 다시 시도해보세요. 문제가 계속되면 ghkrdlsdydapdlf@gmail.com으로 문의해주세요.
지원하는 브라우저는 무엇인가요? +
Chrome, Firefox, Safari, Edge 등 모던 브라우저를 모두 지원합니다. 최적의 성능을 위해 최신 버전 사용을 권장합니다.

🛡️ 보안 및 안전

개인정보는 안전한가요? +
회원가입이 필요 없는 서비스이므로 개인정보를 수집하지 않습니다. 단, 구글 애널리틱스를 통한 익명의 이용 통계는 수집할 수 있습니다. 자세한 내용은 개인정보처리방침을 확인해주세요.
투자 조언을 제공하나요? +
아니요, 저희는 투자 조언을 제공하지 않습니다. 순전히 정보 제공 목적의 데이터와 교육 콘텐츠만 제공합니다. 모든 투자 결정은 본인의 책임이며, 충분한 검토 후 신중하게 결정하시기 바랍니다.
사기나 피싱 사이트는 아닌가요? +
CryptoPulse(cryptopulse.kr)는 정당한 정보 제공 서비스입니다. 어떠한 투자 유도나 자금 요구를 하지 않으며, 단순히 시장 정보만 제공합니다. 의심스러운 사이트는 항상 주의하시기 바랍니다.
